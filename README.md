# SPI対策アプリ

SPI対策のために勢いで作ったアプリ。
問題と答えを入力すると、フラッシュカード形式で確認することができます。
カーソルを合わせるとカードが裏返り、答えを確認することができます。

---

# URL（デモ）

https://github.com/Kazgon-beta/SPI-app.git

---

# 機能一覧

- 問題と答えの入力、フラッシュカードの生成
- 問題と答えのブラウザ上での保存
- フラッシュカードの描画
- フラッシュカードの削除

---

# 使用技術

- HTML
- CSS
- JavaScript
- Git / GitHub

---

# 工夫した点

- cssでフラッシュカードのように動くよう、様々なサイトを参考にして実装した。
- カードを作成した時と、削除した時はアラートにメッセージを表示できるようにした。
- 万が一JSONが壊れていた時のことを考え、配列をロードする際の関数はtry catch構文を利用した。
